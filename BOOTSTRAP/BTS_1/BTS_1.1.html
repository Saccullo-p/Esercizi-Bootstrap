ESERCIZIO BTS_1.1

A cosa serve Bootstrap?

Bootstrap è uno strumento che aiuta nella creazione di siti web senza dover scrivere tutto dall'inizio.

Cosa devi importare per creare una pagina web con Bootstrap?

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/
4.3.1/css/bootstrap.min.css">

Cos’è un CDN? Che vantaggi comporta rispetto a scaricare Bootstrap sul tuo sito?

Il CDN è un gruppo di server distribuiti in più aree geografiche che velocizza la distribuzione dei contenuti web avvicinandoli di più alle posizioni geografiche degli utenti.

A cosa serve questo attributo del tag meta? (content="width=device-width, initial-scale=1)

Indica al browser come controllare le dimensioni della pagina.

Che differenza c’è tra Container e Container Fluid (prova il container fluid creando una pagina chiamata BTS_1.1.1.html)?

Il Container ha una larghezza fissa per ogni dimensione dello schermo, mentre il Container Fluid si espande per riempire la larghezza disponibile. Pagina BTS_1.1.1
